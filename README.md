# MyWishlist

Proyecto de ejemplo con Next.js 14 (App Router), Tailwind CSS y un Server Action que usa OpenAI.

Rápido inicio:

1. Copia tu clave de OpenAI a `.env.local`:

```
OPENAI_API_KEY=sk-xxxx
```

2. Instala dependencias:

```bash
npm install
```

3. Ejecuta en desarrollo:

```bash
npm run dev
```

Visita `http://localhost:3000`.
